## Submission Notes

### Changes in this version:
- Fixed screen flicker issue when switching stations
- Optimized cache performance (instant display of cached data)
- Improved loading state management
- Enhanced debugging and error handling
- Better UI state transitions

### Testing:
- Tested on iOS 18.0+
- Tested on iPhone 15 Pro, iPhone 14, iPad Pro
- Verified smooth station transitions
- Verified cache performance improvements
- All features working as expected

### APIs used:
- OpenTransport API (transport.opendata.ch) - boat schedules
- OpenWeather API - weather data
- Alplakes API (Eawag) - water temperature and forecasts
- MeteoNews API (Vercel) - water levels and temperature fallback
- CoreLocation - nearest station feature

### Privacy:
- Location data only used locally for nearest station feature
- No data is collected or shared
- All API calls are read-only
